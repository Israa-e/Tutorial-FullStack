# Tutorial Full Stack

A collection of hands-on projects and exercises from a full-stack web development course, covering HTML, CSS, and JavaScript fundamentals through progressively harder mini-projects.

## 📁 Folder Structure

### `HTML/`
Core HTML exercises and small projects:
- **08. Let's build Google** – recreating Google's homepage layout
- **09. Anchor tags** – linking between pages
- **12. Lists** – ordered/unordered list practice
- **13. Build a Personal Website** – a personal portfolio page (Vite-powered)

### `CSS/`
Styling exercises and layout projects:
- **02. Intro to CSS** – first steps with CSS, linking stylesheets, margin, padding, centering, flexbox
- **03. Build a Business Card** – a styled business card layout
- **04. Build a Space Exploration Layout** – a themed landing page
- **05. Build a Birthday Website** – a birthday-themed page
- **06. Align-items** – flexbox alignment practice
- **06. Solo project – hometown homepage** – a homepage built from scratch

### `JavaScript/`
Interactive projects using vanilla JavaScript:
- **01. Build a Counter App** – a simple increment/decrement counter
- **02. Solo Project – Basketball Scoreboard** – a scoreboard app with score tracking ([live demo](https://stellar-marzipan-f6dd5c.netlify.app/))
- **04. Build a Blackjack Game** – a playable Blackjack game (includes a bonus **EmojiFighter** mini-game) ([live demo](https://bejewelled-beignet-efbbd8.netlify.app/))
- **05. Password Generator** – a customizable random password generator ([live demo](https://calm-cupcake-02b30e.netlify.app/))

## 🚀 How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/Israa-e/Tutorial-FullStack.git
   ```
2. Open the folder for the project you want to explore.
3. Open the `index.html` file in your browser to view the result.
4. For projects with a `package.json` (like the Personal Website), install dependencies and run the dev server:
   ```bash
   npm install
   npm run dev
   ```
