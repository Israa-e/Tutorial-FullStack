# Tutorial Full Stack

This repository contains a collection of beginner-friendly HTML and CSS practice projects and lessons.

## Folder Structure

### HTML
Contains basic HTML exercises and mini projects such as:
- Google page layout
- Anchor tags and links
- Lists
- A personal website project

### CSS
Contains CSS exercises and styling practice such as:
- Intro to CSS
- Business card design
- Flexbox practice
- Birthday website project

## How to Use
- Open the relevant folder and browse the lesson folders.
- Open the HTML files in your browser to view the result.
- Use the CSS files to learn and experiment with styling.

## Goals
This project aims to help beginners learn and practice:
- Building web pages with HTML
- Styling pages with CSS
- Creating simple layouts and designs
- Developing a strong foundation for front-end web development
